Representações de Cores

Todas elas são baseadas no formato RGB (Red, Green, Blue):

Representação por nome:

style= "background-color: blue; color: white;"

Representação por código hexadecimal:

style= "background-color: #0000ff; color: #ffffff;"

Representação por código RGB:

Por números, o codigo rgb vai de 0 a 255. Neste <h2> utilizei o codigo:

style ="background-color: rgb(0, 0, 255); color: rgb( 255, 255, 255);"
