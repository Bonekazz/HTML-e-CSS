Tags em HTML

No HTML existem varias marcações("tags"). Cada uma com uma certa propriedade e semântica. Passe o mouse por cima de alguns termos e descubra qual a tag deverá ser utilizada:

  • Parágrafos.
  • Abreviações.
  • Linha horizontal.
  • Quebra de linha.
  • Comentários.
  • Lista Ordenada
  • Item de uma lista
